- **테트리스**: https://wonderful-david.github.io/family/tetris.html

## 📋 기능

### 🎂 생일 캘린더

- 가족들의 생일 관리
- 다가오는 생일 알림
- 생일 정보 수정/삭제
- 관계별 분류

### 🧠 오늘의 퀴즈

- 매일 새로운 퀴즈
- 난이도별 포인트 시스템 (쉬움: 1점, 보통: 2점, 어려움: 3점)
- 퀴즈 기록 관리
- 하루 한 번 제한

### 🎮 틱택토 게임

- 가족과 함께하는 클래식 게임
- 2인 플레이 모드
- AI 대전 모드 (미니맥스 알고리즘)
- 실시간 점수 기록
- 승리 조건 하이라이트 효과
- 자동 새 게임 시작

### 🧩 테트리스

- 클래식 테트리스 게임
- 7가지 테트로미노 블록
- 레벨 시스템 (속도 증가)
- 다음 블록 미리보기
- 고스트 블록 (착지 위치 표시)
- 하드 드롭 기능

## 🎮 게임 조작법

### 테트리스

- **← →**: 좌우 이동
- **↓**: 빠른 하강
- **↑**: 블록 회전
- **스페이스**: 즉시 하강
- **P**: 일시정지

### 틱택토

- **마우스 클릭**: 블록 선택
- **R키**: 새 게임
- **AI 모드**: AI와 1대1 대결
